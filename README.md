# ✨ Descripción

**MiniAmigixV** es una plataforma social moderna desarrollada con **Django**, diseñada para conectar personas de manera inteligente, rápida y segura.  

Combina tecnología moderna, autenticación segura y una experiencia visual atractiva para ofrecer una aplicación innovadora.

---

# 🚀 Características Principales

✅ Inicio de sesión con Google  
✅ Registro de usuarios  
✅ Diseño moderno y responsive  
✅ Interfaz atractiva y dinámica  
✅ Sistema de chat inteligente  
✅ Seguridad integrada  
✅ Navegación rápida y fluida  
✅ Estructura profesional escalable
